Introduction
------------

The Calculator module provides simple arithmetic operations.

Installation
------------

Import the module in your Python code::

    from calculator import add

Quick Start
-----------

Example usage::

    from calculator import add

    result = add(2, 3)
    print(result)  # Output: 5
